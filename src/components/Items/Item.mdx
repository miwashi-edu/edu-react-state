# Item

The `Item` component is a **pure presentational component** that displays a product's `name` and `price`, and offers an "Add" button. It is intended to be enhanced via HOC (e.g. `withItemData`) to receive logic and data.

<Canvas>
    <Story name="Default">
        <Item name="Apple" price={1} onAdd={() => {}} />
    </Story>
</Canvas>

## Props

| Prop   | Type     | Description                         |
|--------|----------|-------------------------------------|
| name   | `string` | Name of the product                 |
| price  | `number` | Price of the product in euros       |
| onAdd  | `func`   | Triggered when the user clicks "Add" |

## Usage Example

// In a real application, the onAdd handler would usually be injected by a parent component or HOC.
// This allows the Item to remain agnostic of where its data and logic come from.

```jsx
<Item name="Banana" price={2} onAdd={() => alert("Added!")} />

```
