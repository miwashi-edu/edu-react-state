# Architecture Overview

This project is structured around composable components with injected behavior using Higher-Order Components (HOCs). The core flow links `Shop`, `Item`, and `Cart` via prop-based contract and `cloneElement` composition.

## Components

### `Cart`

- Renders a floating cart interface
- Can be **controlled** (`open`, `setOpen`) or **uncontrolled** (internal state)
- Requires:
- `items` array
- `onRemove(index)`
- Optionally injects `onAddToCart(item)` into children

```jsx
<Cart items={...} onRemove={...}>
  <Shop />
</Cart>
