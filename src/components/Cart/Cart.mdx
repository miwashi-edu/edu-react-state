# Cart
The `Cart` component wraps your application content and displays a cart interface that can be toggled or minimized.

<Canvas>
    <Story name="Default">
        <Cart items={[{ name: 'Apple', price: 1 }, { name: 'Banana', price: 2 }]} onRemove={() => {}} >
            <div>Shop content goes here</div>
        </Cart>
    </Story>
</Canvas>

## Usage

```jsx
import withLocalStorage from './withLocalStorage';
import Cart from './Cart';
import Shop from '../Shop';

const LocalCart = withLocalStorage(Cart);

const App = () => {
  const handleAddToCart = (item) => {
    // Cart receives this through clone injection
    // and stores item.id via withLocalStorage
  };

  return (
    <LocalCart>
      <Shop onAddToCart={handleAddToCart} />
    </LocalCart>
  );
};
